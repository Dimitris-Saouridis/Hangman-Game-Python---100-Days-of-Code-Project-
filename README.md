# 🎮 Hangman Game (Python)

A simple **Hangman game** built in Python as part of the **"100 Days of Code - Python Bootcamp" by Udemy**, with additional personal improvements.

---

## 📌 Features

* Random word selection
* Lives system (6 attempts)
* ASCII art for hangman stages
* Tracks correct and wrong guesses
* Prevents duplicate guesses
* Clean and simple game logic

---

## 📁 Project Structure

```
hangman-game/
│
├── main.py              # Main game logic
├── hangman_art.py      # ASCII art (logo + stages)
├── hangman_words.py    # Word list
```

---

## ▶️ How to Run

1. Make sure you have Python installed
2. Open terminal in the project folder
3. Run:

```bash
python main.py
```
